# `vhost`
`vhost` helps you programitcally control your apache virtual hosts via the command-line (soon!) or a smart REST API.

> NOTICE: `vhost` is in beta. Use at your own risk.

## Usage

``` 
const { vhost, ssl } = require('vhost')
```
